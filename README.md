# 嶋水道工事店｜営業提案用デモサイト

嶋水道工事店様へご提案するための、1ページ型の静的デモサイトです。`noindex, nofollow` を設定しており、公式サイトではありません。

## 編集箇所

- 本文・連絡先・リンク: `site/index.html`
- デザイン: `site/styles.css`
- メイン画像: `site/assets/hero-plumbing-reference.png`（生成した施工参考イメージ）

## ローカル確認

`site` ディレクトリを静的サーバーで開きます。例: `npx serve site`

## Cloudflare Pages

GitHubリポジトリを接続し、Framework preset は `None`、Build command は空欄、Build output directory は `site` を指定します。
